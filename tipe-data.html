Tipe Data

  • Tipe Data ==> jenis data yang dapat disimpan oleh variabel / wadah
  • Jenis / Macam Tipe Data:
    • String / teks ==> data berupa karakter / kata / kalimat
    • Number / bilangan ==> data berupa angka (bilangan bulat / desimal)
    • Boolean ==> datanya hanya terdiri atas: true atau false
    • Array ==> dapat menampung 1 atau lebih data sekaligus
    • Object
