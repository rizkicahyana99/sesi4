Variabel

  • Variabel ==> wadah / tempat / media untuk menyimpan data sementara
  • Variabel harus dibuat / dideklarasikan terlebih dahulu sebelum digunakan
  • Keyword / kata kunci deklarasi variabel:
    • Var ==> untuk nilai yang berubah-ubah ==> old ways
    • Let ==> untuk nilai yang berubah-ubah
    • Const ==> konstanta ==> untuk nilai yang tetap / fix / tidak bisa berubah
